## Ambrosio
Por ser uma API de Laudos e Exames, algo relacionado a medico, doutores e medicina, o nome Ambrosio foi escolhido em homenagem a Sando Ambrosio, Santo de **Doutor** da igreja

![image](https://user-images.githubusercontent.com/5832193/64138306-e9273d80-cdd2-11e9-8382-2f5cbcc974b6.png)

## How to install
### Requisites
- Java 11
- Docker
- Docker Compose
- Maven
```
git clone 
sudo docker-compose up -d
```

### Tests
```mvn clean compile test```
